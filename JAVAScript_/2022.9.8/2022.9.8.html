Javascript - 비교(2/4):== 과 ===

==:동등 연산자로 좌향과 우향을 비교해서 서로 값이 같다면 true 다르다면 false 가 된다

예 시
alert(1==2)
//false
alert(1==1)
//true
alert("one"=="two")
//false
alert("one"=="one")
//true

===(equal operator): 일치 연산자로 === 좌항과 우항이 '정확'하게 같을 때 true 다르면 false가 된다. 여기서 정확하다는 말의 의미에 집중하자. 아래 예를보자.

예시
alert(1=='1');
//true
alert(1==='1');
//false

Javascript - 비교(3/4) ===를 사용하자 !

예시
alert(null == undefined) //true
alert(null === undefined); //false
alert(true == 1); //true
alert(true === 1); //false
alert(true == '1'); //true
alert(true === '1'); //false
alert(0 === -0); //true
alert(NaN === NaN); //false
null : 값이 없는것 undefined: 정의되지 않은 것

Javascript - 비교(4/4) 부정과 부등호

> :좌향이 우향보다 크다면 참 , 그렇지 않다면 거짓임을 알려주는 연산자다
>=: 좌향이 우향보다 크거나 같다. '<=는 반대의 의미로 언급은 생략

*Javascript - 조건문(1/5): 조건문이란 *

-조건문이란: 주어진 조건에 따라서 에플리케이션을 다르게 동작하도록 하는 것이다.
if: 뒤의 괄호에 조건이 오고 , 조건이 될 수 있는 값은 boolean이다
예 제
아래 예제의 실행결과는 'result : true'다. if 뒤에 true가 왔기 때문이다.
if(true){
alert('result : true');
}
-------------------------------------------------------------------------------
다음 예제는 아무것도 출력하지 않을 것이다. if 뒤에 false가 왔기 때문이다.
if(false){
alert('result : true');
}
