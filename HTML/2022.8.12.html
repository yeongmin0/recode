  • HTML - form : method

    •  -post:url로 전달할떄 감취서 전달하는 방식을 말한다
    *폼 태그를 사용한다고하면 메소드(method>를 포스트(post)방식으로 처리하면 된다



    ----- 예 시 코 드
    < form action="http:localhost/method.php" method="post">
    < input type="txet" name="id">
    < input type="password" name="pwd">
    < input type="submit">
    < /form>
