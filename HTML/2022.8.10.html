    •

    -value:페이지를 열었을때 기본적으로 적혀있는 태그
    -taxtarea:여러 줄을 입력 받을때 사용하는 태그
    -< input type="txet" >
    -< input type="password" >
    -< textarea >

    여러줄을 입력하기 어려울때 별도의 < textarea > 이 태그의 기본값을 그 태그의 안쪽에다가
    컨텐츠에 입력 할 수 있게끔 다른 형식의 문법을 가지고 있는 것을 알아두면 좋다

    < /p >

  • HTML - form: radio,checkbox

    •  -radio 버튼은 네임(name) 값이 같은 것들 끼리 그루핑이 되는것이고
    같은 그룹에속하는 것 중에 하나가 선택되면 나머지의 선택이 해제되는 형태의 버튼
    -checkbox:라디오 버튼과는 다르게 같은 네임을 가지고 있는 여러개의 컨트롤들을 다중으로 선택할 수 있다
    -checked:기본적으로 선택 되기를 원한 속성없는 태그

    선택 1 빨간색: 검은색: 파란색:

    선택 2

    25: 50: 100:


    -----예시 코드
    < p> < form accept="http://locahost./order.php"
    < h1>선택 1
    빨간색:< input type="radio" name="color" value="red">
    검은색:< input type="radio" name="color" value="black">
    파란색:< input type="radio" name="color" value="blue">
    < /p>

    < h1>선택 2
