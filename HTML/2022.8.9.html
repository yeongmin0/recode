    대부분 사용자들은 릴리즈 바이너리를 설치하므로 마이크로소프트의 독자적 라이선스 하에 VS Code를 사용하고 있다.

    -< td /td >= 테이블 데이터 (table data) 표의 각각의 데이터를 의미하는 것
    -< tr /tr >= 테이블 로우 (table row) 같은 행에 속하는 td 태그들을 tr 이라는 태그로 묶어주면
    웹 브라우저는 tr 아래에 있는 td들을 같은 행으로 표시
    -< table /table >= 묶어주는 용도
    -< table border="" >= 테두리 (숫자에 따라 두께가 달라짐/ "css라는 언어로 별도로 꾸며야 됨")
    -----
    table border="2"
    < tr >
    < td >이름< /td > < td>성별 < td>주소
    < /tr >
    < tr >
    < td >강영민< /td> < td>남< /td> < td>인천< /td>
    < /tr>
    < /table>
    -----
