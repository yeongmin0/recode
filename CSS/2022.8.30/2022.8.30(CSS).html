4가지의 포지션

  1. 스택티 (static) 기본
  2. 렐렅티브 (relative) 상대적인 크기
  3. 앱설루트 (absolute) 절대적인 크기
  4. 픽시트(fixed)는 스크롤과는 상관없이 화면에 고정 시키는 것
  5. 그 중에 앱설루트 (absolute)와 픽시트(fixed) 는 부모와 링크가 끊기기 때문에 부피가 자기의 컨텐츠 크기만해지고
    그리고 부모 엘리먼트들은 자식을 없는 셈치는, 그래서 작아지는 특징이 있다는 것

CSS 수업 - 포지션 2 : relative

포지션 2 :relative 실습 자료

style>
#parent,#other{border: 5px solid red;
}
#me{
background-color: black;
color: white;
position: absolute;
}
.vip{
position: relative;
top: 200px;
}


< body>
< div id="other">other
< div id="parent">
parent
< div id="me">me
< /div>
< div class="vip">본인이 left , top , offset 을 사용하기 위해서는 position relative 포지션 타입을 지정 할 핑요가 있다
그리고 "포지션 타입을 지정하지 않았다" 는 static 이라고 한다
